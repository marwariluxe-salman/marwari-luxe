# Favicon Files

Please add the following favicon files to your `public` folder:

## Required Files:
- `favicon.ico` (16x16, 32x32 multi-size)
- `favicon-16x16.png`
- `favicon-32x32.png` 
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

## How to Generate:
1. **Online Generator**: Use https://realfavicongenerator.net/
2. **Upload your logo** (preferably 512x512 PNG)
3. **Download the generated pack**
4. **Place all files** in the `public` folder
